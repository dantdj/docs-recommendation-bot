# docs-recommendation-bot
Slackbot intended to respond to "how do I?" style messages with suggested docs

## Running the application

This app leverages `pipenv` ([see here](https://pipenv.pypa.io/en/latest/)). Packages are managed through the Pipfile. As a result, `pipenv` must be used to run the file.

To launch the application, run `pipenv run python app.py`.
